Que se passe-t-il si le centre de service ne peut pas réparer le produit pendant la période de garantie?

1.Emballer l'article

  • Si les marchandises ne sont pas réparables ou si, pour une raison quelconque, elles ne peuvent pas être réparées dans les délais impartis, le centre de service agréé peut délivrer un certificat de radiation.
  • Vous pouvez nous contacter avec cet acte, et nous serons en mesure d'échanger les marchandises ou d'effectuer un remboursement sur la base de cet acte.

  • L'acte doit contenir les informations suivantes:

    • votre nom et prénom;
    • le nom du centre de service;
    • le modèle du produit;
    • les défauts revendiqués, les défauts identifiés lors de l'examen;
    • la raison pour laquelle les marchandises ne peuvent pas être réparées.
  • Il indique également où les marchandises restent après que le certificat vous a été délivré : dans le centre de service ou si elles vous sont remises.

  • Pour pré-vérifier la documentation, veuillez envoyer un paquet de documents à l'adresse e-mail act?rozetka.com.ua, qui comprend :

    • copies scannées (photos) des documents fournis par le centre de service - loi sur le centre de service;
    • photo, carte de garantie;
    • photo du ticket de caisse (facture);
    • photo du produit avec le numéro de série.
  • N'oubliez pas d'indiquer dans l'objet du message : Acte pour inspection.
  • Après vérification, notre responsable vous contactera pour convenir des mesures à prendre.
